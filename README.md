# Amped Contracts
Contracts for Amped.

## Install Dependencies
If npx is not installed yet:
`npm install -g npx`

Install packages:
`npm i`

## Compile Contracts
`npx hardhat compile`

## Run Tests
`npx hardhat test`

*** npx hardhat run --network pegasus scripts/deploy.js
*** npx hardhat verify --network pegasus [address]
